§3　解析的証明
3.1　u(0) < 0 の構造
各レベル L_i への入力 x_{i-1}^out と下限 φ^i の関係を調べる。
L1（i=1）：初期入力 x_0 = φ
u_1(0) = φ − φ^1 = φ − φ = 0　→　境界値からスタート
L_i（i≥2）：前レベル出力 x_{i-1}^out ≈ φ^{i-1}
u_i(0) = φ^(i−1) − φ^i = φ^(i−1)(1 − φ) = −φ^(i−2) < 0
すなわち L2 以降は全て「範囲の外（下）」から入力される。 これが本証明の核心。
3.2　u(0) < 0 時の動力学
x < φ^i（範囲外）では Tower項 = 0。方程式は：
du/dt = −e·u − τ·u³
u < 0 のとき du/dt = −e·u − τ·u³ > 0（uが負なので両項とも正）。指数的に u → 0 へ引き寄せられる。
線形近似：
u(t) ≈ u(0) · exp(−e · t)

§4　φが出現する理由
4.1　幾何的根拠
区間境界を φ^i というφ等比スケールで設定したことが直接の原因。等比スケールにより：
δ_i = φ^(i+1) − φ^i = φ^i(φ−1) = φ^i · (1/φ) = φ^(i−1)
各レベルの幅 δ_i 自体がφのべき乗。そして：
x_i^out ≈ φ^i = φ^(i+1) / φ  ⟹  end/high = 1/φ
φ等比スケールで区間を切った瞬間に、1/φ比は構造として埋め込まれていた。
4.2　w/δ の一定性
全レベルで Tower係数と区間幅の比が一定：
w_i / δ_i  ≈  3.5789　（全L1〜L8で不変）
これは w_i = 3.5789 · δ_i = 3.5789 · φ^(i−1) という設計（φ完全表現）の帰結。スケールが変わっても動力学の「形」が保たれる自己相似構造。
4.3　SUTの文脈での意味
「繋げたらφに落ちた」のではなく、「φスケールで区間を切った時点でφは埋め込まれていた」。 発見ではなく導出。SUT「背理還流」の構造的表現。
